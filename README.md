# 🎮 Análisis de Datos de Videojuegos

## 📌 Descripción del proyecto

Este proyecto consiste en un análisis exploratorio de datos (EDA) sobre la industria de los videojuegos, con el objetivo de identificar los factores que influyen en el éxito comercial de un videojuego.

Se trabajó con un dataset que incluye información sobre ventas, plataformas, géneros, calificaciones de usuarios y críticos, así como datos regionales.

---

## 🎯 Objetivo

Analizar el comportamiento del mercado de videojuegos para:

- Identificar tendencias de ventas
- Detectar plataformas y géneros más exitosos
- Evaluar el impacto de las reseñas en el desempeño comercial
- Obtener insights que apoyen la toma de decisiones en la industria

---

## 📊 Dataset

- Más de **16,000 registros**
- Variables principales:
  - Nombre del juego
  - Plataforma
  - Año de lanzamiento
  - Género
  - Ventas globales y por región (NA, EU, JP)
  - Calificaciones de usuarios y críticos

---

## 🛠️ Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 🔍 Proceso de análisis

1. Limpieza y preparación de datos
2. Análisis exploratorio (EDA)
3. Visualización de datos
4. Identificación de patrones y tendencias
5. Generación de insights

---

## 📈 Principales insights

- Algunas plataformas dominan el mercado en diferentes periodos de tiempo  
- Los géneros de acción y deportes suelen tener mayor volumen de ventas  
- Existe correlación entre calificaciones de críticos y éxito comercial  
- Las preferencias de los usuarios varían significativamente por región  

---

## 🚀 Conclusión

El análisis permitió comprender mejor los factores que influyen en el éxito de un videojuego, proporcionando información útil para la toma de decisiones estratégicas en la industria.
